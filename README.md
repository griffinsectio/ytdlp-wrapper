# ytdlp-wrapper
A command line wrapper for ytdlp to simplify the usage of yt-dlp.
Currently suppported platforms:
- [X] **Linux**
- [ ] **Windows**
- [ ] **BSD**
# Installation
## Linux
```
chmod +x ./install.sh
sudo ./install.sh
```
